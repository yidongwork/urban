Extrair Dados

Com esta ferramenta, pode selecionar e descarregar dados relativos a uma área de interesse definida. As camadas selecionadas são adicionadas a um ficheiro zip ou a um pacote de camadas.

Camadas a Extrair

Camadas na tabela de conteúdos a partir das quais pode extrair elementos. As camadas assinaladas são extraídas e incluídas num ficheiro zip ou num pacote de camadas resultante.

Área de estudo

Área que define os elementos que serão incluídos no ficheiro zip ou no pacote de camadas resultante. Pode selecionar uma das seguintes opções:

  • Extensão atual visível do mapa ( Igual à Visualização).
  • Uma das camadas incluídas na tabela de conteúdos.
  • Desenhar uma área para definir a área de estudo.
Serão extraídos todos os elementos que se encontrem totalmente dentro da extensão do mapa. No caso de elementos que cruzam ou intersetam o limite da extensão da mapa, mas não se encontram totalmente dentro da extensão, dispõe de duas opções:
  • Selecionar elementos – Serão extraídos todos os elementos que cruzam o limite da área de estudo.
  • Cortar elementos – Serão extraídas apenas as secções dos elementos que se encontram dentro da área de estudo.

Tipo de dados de saída

Formato dos dados que serão extraídos e descarregados. Os pacotes de camadas incluirão sempre file geodatabases.

Nome do ficheiro de saída

Este é o nome do item que será adicionado a O Meu Conteúdo. O nome do item é baseado no Formato dos dados de saída selecionado. Se o item já existir, ser-lhe-á pedido que confirme se pretende substituir o item.

Utilizando a caixa de lista pendente Guardar resultado em, pode definir o nome de uma pasta em O Meu Conteúdo para guardar o resultado.
