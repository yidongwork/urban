Melhorar Camada

Esta ferramenta melhora os dados de pontos ou áreas obtendo factos relacionados com as pessoas, os locais e os negócios que estão à volta das localizações dos dados. A ferramenta Melhorar Camada permite responder a novas perguntas sobre localizações que não é possível responder apenas com mapas. Por exemplo: Que tipo de pessoas vive aqui? O que é que as pessoas gostam de fazer nesta área? Quais são os seus hábitos e estilos de vida? Que tipo de negócios existe nesta área?

O resultado será uma nova camada que contém toda a informação demográfica e geográfica proveniente de determinadas coleções de dados. Essa nova informação é adicionada como campos na tabela.

Se marcar a opção Utilizar a extensão atual do mapa, serão analisados apenas os elementos incluídos na camada de análise e de área visíveis na extensão atual do mapa. Se não marcar a opção, serão analisados todos os elementos incluídos na camada de análise e na camada de área, mesmo que estejam fora da extensão atual do mapa.

Selecionar coleção de dados

Selecione a região pretendida para obter uma lista das coleções de dados disponíveis que pretende utilizar para melhorar os dados. No caso dos Estados Unidos, do Canadá e muitos outros países em todo o mundo, pode melhorar os dados adicionando diversas coleções de dados com listas de atributos pré-elaboradas que fornecem informação demográfica e contexto geográfico relativos a pessoas, locais e negócios que estão à volta das localizações dos dados.

Áreas a melhorar

Estas são as áreas que pretende melhorar com informação adicional contida na coleção de dados. Se a camada de análise tiver pontos, são criadas áreas à volta de cada ponto utilizando uma distância em linha reta, uma distância de condução ou um tempo de condução.

Nome da camada de resultado

Este é o nome da camada que será criada em O Meu Conteúdo e adicionada ao mapa. O nome padrão é baseado no nome da camada de análise. Se a camada já existir, ser-lhe-á pedido que confirme se pretende substituir a camada.

Utilizando a caixa de lista pendente Guardar resultado em, pode definir o nome de uma pasta em O Meu Conteúdo para guardar o resultado.
