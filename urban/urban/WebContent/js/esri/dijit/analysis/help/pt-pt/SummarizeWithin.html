Resumir Dentro

Esta ferramenta localiza elementos (e partes de elementos) que se encontram dentro dos limites de áreas na camada de análise. Por exemplo:

  • Tendo em conta uma camada de limites de separação de água e uma camada de limites de utilização de terreno por tipo de utilização, calcular a área total de tipo de utilização de terreno por cada separação de água.
  • Tendo em conta uma camada de parcelas num concelho e uma camada de limites urbanos, resumir o valor médio de parcelas vazias dentro de cada limite urbano.

Se marcar a opção Utilizar a extensão atual do mapa, serão resumidos apenas os elementos da camada de análise e da camada que pretende resumir que estão visíveis na extensão atual do mapa. Se não marcar a opção, serão analisados todos os elementos da camada de análise e da camada que pretende resumir, mesmo que estejam fora da extensão atual do mapa.

Choose layer to summarize

Serão resumidos os elementos desta camada que estejam dentro dos limites de elementos incluídos na camada de entrada.

Add statistics from

Opcionalmente, pode calcular estatísticas relativas aos atributos numéricos dos pontos que se encontram dentro dos limites de elementos.

Se marcar a caixa de seleção Contagem de pontos, pode calcular o número total de pontos dentro de cada limite de área. Adicionalmente, pode calcular estatísticas relativas aos atributos numéricos dos pontos que se encontram dentro de cada limite de área.

  • Soma – Valor total do campo definido.
  • Média – Valor médio do campo definido.
  • Mínimo – O valor mais pequeno no campo definido.
  • Máximo – O valor maior no campo definido.
  • Desvio Padrão – Desvio padrão dos valores no campo definido.

Se marcar a caixa de seleção Comprimento total das linhas em, pode calcular o comprimento total das linhas dentro de cada limite de área. Adicionalmente, pode calcular estatísticas relativas aos atributos numéricos das linhas que se encontram dentro de cada limite de área.

Add statistics from

Opcionalmente, pode calcular estatísticas relativas aos atributos numéricos das áreas próximas.

Se marcar a caixa de seleção Soma da área utilizando, pode calcular a área total dos elementos que se encontram dentro de cada limite de área da camada de análise. Adicionalmente, pode calcular estatísticas relativas aos atributos numéricos dos elementos que se encontram dentro de cada limite de área.

  • Soma – Valor total do campo definido.
  • Média – Valor médio do campo definido.

  • Máximo – O valor maior no campo definido.
  • Desvio Padrão – Desvio padrão dos valores no campo definido.

Choose attribute to group by

Este é um atributo dos elementos Resumir que pode utilizar para calcular estatísticas separadamente para cada valor de atributo único. Por exemplo, suponha que a camada de análise contém limites urbanos e os elementos Resumir são parcelas. Um dos atributos das parcelas é Status, que contém dois valores: VAZIA e OCUPADA. Para calcular a área total (ou qualquer outro atributo, como Value) de parcelas vazias ou ocupadas dentro dos limites de cidades, utilize Status como atributo de agrupamento.

Result layer name

Este é o nome da camada que será criada em O Meu Conteúdo e adicionada ao mapa. O nome padrão é baseado no nome da camada de análise. Se a camada já existir, ser-lhe-á pedido que confirme se pretende substituir a camada.

Utilizando a caixa de lista pendente Guardar resultado em, pode definir o nome de uma pasta em O Meu Conteúdo para guardar o resultado.
