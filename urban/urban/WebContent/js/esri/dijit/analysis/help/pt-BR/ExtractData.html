Extrair Dados

Com esta ferramenta, você pode selecionar e baixar dados para uma área de interesse específica. As camadas que você selecionar serão adicionadas em um pacote de camada ou arquivo ZIP.

Camadas para Extrair

As camadas em sua área de controle das quais você pode extrair feições. As camadas que estão selecionadas são extraídas e incluídas em um pacote de camada ou arquivo ZIP resultante.

Área de Estudo

A área que define quais feições serão incluídas no pacote de camada ou arquivo ZIP de saída. Você pode escolher do seguinte:

  • A extensão visível atual do seu mapa ( Mesmo da visualização).
  • Uma das camadas em seu índice.
  • Desenhar uma área para definir a área de estudo.
Todas as feições completamente dentro da extensão de mapa serão extraídas. Para feições que cruzam ou intersectam o limite da extensão de mapa, mas não estão completamente dentro da extensão, você tem duas opções:
  • Selecionar feições—Feições inteiras que cruzam o limite da área de estudo serão extraídas.
  • Recortar feições—Somente estas porções de feições que estão dentro da área de estudo serão extraídas.

Tipo de Dados de Saída

O formato dos dados que serão extraídos e baixados. Os pacotes de camada sempre incluirão arquivo geodatabases.

Nome do Arquivo de Saída

Este é o nome do item que será adicionado em Meu Conteúdo. O nome do item é baseado no Formato dos dados de saída escolhido. Se o item já existir, você será solicitado para confirmar se deseja substituí-lo.

Utilizando a caixa suspensa Salvar resultado em, é possível especificar o nome de uma pasta em Meu Conteúdo onde o resultado será salvo.
