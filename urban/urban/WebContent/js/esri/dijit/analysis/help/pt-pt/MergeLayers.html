Juntar Camadas

Esta ferramenta copia elementos de duas camadas para uma nova camada. As camadas a juntar devem conter os mesmos tipos de elementos (pontos, linhas ou áreas). Pode controlar o método de juntar e copiar os campos das camadas de entrada. Por exemplo:

  • Tenho três camadas para Inglaterra, País de Gales e Escócia e quero obter uma só camada de Grã-Bretanha.
  • Tenho 12 camadas e cada uma contém informação relativa a parcelas de freguesias adjacentes. Quero juntá-las numa só camada, mantendo apenas os campos que têm o mesmo nome e o mesmo tipo nas 12 camadas de entrada.

Se marcar a opção Utilizar a extensão atual do mapa, serão combinados apenas os elementos visíveis na extensão atual do mapa. Se não marcar a opção, serão combinados todos os elementos incluídos nas duas camadas, mesmo que estejam fora da extensão atual do mapa.

Choose Layer

Selecione a camada que pretende juntar à camada de análise.

Modify merging attributes

Por predefinição, todos os campos de ambas as camadas de entrada são transferidos para a camada de saída. Pode alterar a forma de transferir os campos para a camada de saída da seguinte forma:

  • Renomear – O nome do campo será mudado na camada de saída.
  • Remover – O campo será removido da camada de saída.
  • Corresponder – É dada equivalência aos nomes dos campos. Os valores do campo da camada que pretende juntar serão copiados para o campo de saída correspondente.

Result layer name

Este é o nome da camada que será criada em O Meu Conteúdo e adicionada ao mapa. O nome padrão é baseado no nome da camada de análise. Se a camada já existir, ser-lhe-á pedido que confirme se pretende substituir a camada.

Utilizando a caixa de lista pendente Guardar resultado em, pode definir o nome de uma pasta em O Meu Conteúdo para guardar o resultado.
